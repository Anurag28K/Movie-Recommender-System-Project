# 🎬 Movie Recommender System

This is a Content-Based Movie Recommendation System built using Python and Streamlit.

## 🚀 Features
- Select a movie from the dropdown
- Get top 5 similar movie recommendations
- Movie posters fetched using TMDB API
- Simple and interactive web interface

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- TMDB API

## 📂 Project Structure
- app.py → Main Streamlit Application
- similarity.pkl → Precomputed similarity matrix
- movie_dict.pkl → Movie dataset dictionary

## ▶️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/yourusername/Movies-Recommender-System.git

2. Go inside the folder:
   cd Movies-Recommender-System

3. Install requirements:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app.py

## 📌 Note
Similarity matrix and dataset files are required to run the project.
